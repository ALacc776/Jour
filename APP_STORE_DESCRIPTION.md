## App Review Notes (for Apple Reviewers)

### Demo Instructions
The app is fully functional upon first launch. No account or setup required.

**To test all features:**
1. Add entries using the text field on Today tab
2. Tap "+ Add" on Calendar tab to see time-based prompts
3. Test camera/photo features (requires camera/photo permissions)
4. Go to Settings (gear icon on Today tab) to:
   - Enable notifications
   - Export data
   - View Privacy Policy
   - Access Support

**Quick Entry Feature:**
- Tap "+ Add" on Calendar
- Select "Quick entry"
- Type multiple lines (each line becomes a separate entry)
- Tap "Save X entries"

**Export Feature:**
- Go to Settings
- Scroll to "Export to Clipboard"
- Test "Copy Today", "Copy This Week", etc.
- Paste in Notes to see formatted output

### Permissions Used:
- **Camera**: To take photos for journal entries (optional)
- **Photo Library**: To select existing photos (optional)
- **Notifications**: For daily journaling reminders (optional, user-controlled)

All permissions are optionalâ€”core journaling works without any permissions.

### Privacy:
All data is stored locally using UserDefaults and local file storage. No network requests are made. No analytics or tracking. This can be verified by network monitoring during review.
